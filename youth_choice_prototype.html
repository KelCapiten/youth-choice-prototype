RFP Alignment: Technical & Structural Architecture

1. Low Bandwidth PWA

Designed to function perfectly with high-latency, expensive data packets in Balaka and Lilongwe. Custom service-worker captures resource queries locally and interfaces with IndexedDB storage queue. Assets like local maps use vector coordinate representations requiring less than 85KB overhead.

2. Malawi DPA Compliance

Under Section 17, processing minor's health data strictly mandates guardian consent. Our framework implements a safe tier-based onboarding, which dynamically separates non-PII educational access from personal premium modules to guarantee safety.

3. Cross-Channel ETL

The BI dashboard acts as a single point of truth. It consolidates database syncs, WhatsApp webhooks, Facebook Graph parameters, and daily dial-logs from Chipatala cha pa Foni (CCPF) to let decision-makers analyze health trends in real time.
